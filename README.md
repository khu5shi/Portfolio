# 🌐 Personal Portfolio

A modern and responsive **portfolio website** built with **React**, powered by **Vite** for lightning-fast performance.  
Styled using **Tailwind CSS** for a clean UI and enhanced with **Framer Motion** for smooth animations and interactions.  

---

## ✨ Features
- ⚡ **Fast & Optimized** – Vite ensures blazing-fast builds & HMR.  
- 🎨 **Modern UI** – Responsive and elegant design with Tailwind CSS.  
- ✨ **Smooth Animations** – Engaging transitions powered by Framer Motion.  
- 📱 **Fully Responsive** – Works perfectly on desktop, tablet, and mobile.  
- 🛠 **Easy Customization** – Well-structured and developer-friendly code.  

---

## 🛠 Tech Stack
- **React** – Component-based architecture  
- **Vite** – Next-gen frontend tooling  
- **Tailwind CSS** – Utility-first styling  
- **Framer Motion** – Animations & transitions
